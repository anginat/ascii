ASCII Table

ASCII short for American Standard Code for Information Interchange, is a character encoding standard for communication in electronic devices.

ASCII codes in the below table represent symbols in form of various characters in computers, telecommunications equipment, and other devices.

Most modern character-encoding schemes are based on ASCII, although they support many additional characters.
