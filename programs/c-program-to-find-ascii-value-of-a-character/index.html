C Program to find ASCII value of a Character

This C program is used to find and print the ASCII value of a letters, numbers, and other characters.

Each character or number has its own ASCII value.

Example:

The ASCII value of the number5' is 53and for the letter'A' is 65.

Program:
/* 
* Created on March 02, 2020, 12:45 PM
*/ 

#include 

int main() {
char c = 'w';
printf("ASCII value of %c is %d\n", c, c);
return 0;
}
Output: 
ASCII value of w is 119

The following program accepts character as input from the user and then output the ASCII code.

Program with stdin:
/* 
* Created on March 02, 2020, 12:47 PM
*/ 

#include 

int main() {
char c;
printf("Please enter a character: ");
scanf("%c", &c); 
printf("ASCII value of %c is %d\n", c, c);
return 0;
Stdin:
Please enter a character: g
Output:
ASCII value of g is 103
